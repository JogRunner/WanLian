  • 2. 负责公司产品UI界面、图片等设计

  • 4. 负责公司各类宣传物料（产品宣传页、手册，平面海报设计，展架等）的策划与设计
  • 5. 负责为公司日常的运营活动提供美术支持
  • 6. 负责完成公司策划方案的排版或PPT模板设计、美化工作
  • 7. 配合相关部门完成其它分配的工作
    岗位要求：
  • 1. 专科或以上学历，计算机、美术或设计等相关专业，有良好美工功底
  • 2. 具有一年以上平面设计经验，熟悉VI设计与维护
  • 3. 精通Photoshop、Illustrator、Indesign等平面设计软件，掌握Dreamweaver等制作工具
  • 4. 思维活跃，富有创意，有较强的沟通能力
  • 5. 具备良好合作态度及团队精神，工作富有激情，抗压性强

P.S. 简历中请附相关作品

移动互联网产品经理

工作职责：
  • 1. 根据公司策略及客户需求进行手机客户端产品的规划设计和产品线进程管理，包括：功能，特性，使用流程，UI/UE，性能要求等
  • 2. 负责配合开发人员进行开发工作，推动及协调产品的开发进度，把控产品质量，完成测试、改进、直到上线
  • 3. 提供产品原型设计文档和商用文档，包括MRD、PRD流程图、白皮书、产品使用手册、FAQ等
  • 4. 通过市场调查和运营数据分析，对产品发展情况进行分析、评估，优化流程，对各阶段的产品功能和特性进行取舍
  • 5. 了解移动互联网领域的相关、类似产品，分析行业发展热点，并提出创新或改进需求
    • 岗位要求：
  • 1. 大学本科以上学历，熟悉电子商务行业，本地生活服务，对O2O业务模式及行业未来趋势有深刻认识
  • 2. 熟悉Android、iOS操作系统、微信公众号平台特性，对交互设计及用户体验有浓厚的兴趣
  • 3. 具有良好的学习能力
  • 4. 良好的书面及口头表达能力，注重细节，能承受较大压力
  • 5. 独立创新能力强，良好的逻辑思维能力，注重细节，执行力强，工作态度严谨

数据挖掘工程师

工作职责：
  • 1. 负责对用户特征提取，用户聚类分类，推荐系统协同过滤等业务问题，提供先进、适用的解决方案（模型）及高可扩展性的可并行化算法设计
  • 2. 协助对项目实施中遇到的算法性能瓶颈进行诊断和改进
  • 3. 编写脚本及创建统计分析系统，实现对海量业务数据的统计分析
  • 4. 利用并改进数据挖掘、人工智能等分析模型，发掘用户行为中蕴藏的潜在规律
  • 5. 以数据为基础，实现符合用户需求的产品智能推荐，用户行为分析等
岗位要求：
  • 1. 熟练运用各种常用算法和数据结构，有独立的实现能力
  • 2. 熟悉 Linux平台上的编程环境，精通 C++/Java开发，精通 Python/Shell等脚本语言
  • 3. 有文本分析、自然语言处理、推荐系统或者大数据分析工作经验
  • 4. 具有丰富的数据仓库开发经验，对数据处理、数据建模、业务理解有深刻认识
符合以下条件优先：
  • 1. 英文技术文献阅读和理解能力
  • 2. 熟悉分布式系统的设计和应用，熟悉分布式、缓存、消息等机制
  • 3. 思路清晰，主动性强，有很强的自我驱动力

  • 3. 有OpenWrt开源应用贡献者及有流控开发经验者优先
  • 4. 爱钻研、踏实，能承受一定工作压力

市场拓展专员

  • 2. 组织和监督实施年度市场推广计划
  • 3. 进行市场调研与分析，研究同行、业界发展状况，定期进行市场预测及情报分析，为公司决策提供依据
  • 4. 制定公司整体公关策略及危机公关的应对处理

  • 7. 管理市场团队，并对团队成员和相关部门进行市场培训和指导
岗位要求：
  • 1. 市场营销管理类或相关专业大专以上学历
  • 2. 三年以上市场营销工作经验，有商场、店商方面销售经验，对该领域发展有深刻理解
  • 3. 具备很强的策划能力，熟悉各类媒体运作方式，有大型市场活动推广成功经验
  • 4. 具有敏感的商业和市场意识，分析问题及解决问题能力强，具有优秀的资源整合能力和业务推进能力
  • 5. 具备良好的沟通合作技巧及丰富的团队建设经验
