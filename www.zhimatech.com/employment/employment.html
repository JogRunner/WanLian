运营专员

工作职责：
  • 1. 负责公司硬件产品（路由设备）的订单处理、发货及库存管理工作
  • 2. 负责购买了公司产品的客户的服务开通工作
  • 3. 负责接听客户的反馈电话，及时记录下问题，并反馈相关部门，跟进问题的处理与解决
  • 4. 负责制作整理每周公司运营情况的报告
    岗位要求：
  • 1. 大专以上学历，2-3年工作经验，有相关（B端）客户服务经验
  • 2. 工作态度积极、努力、认真，能有效处理繁琐细致的工作
  • 3. 吃苦耐劳
  • 4. 沟通能力较强，能有效与他人沟通协作
  • 5. 积极进取，学习意愿强

出纳

工作职责：
  • 1. 配合各部门办理报销等有关手续
  • 2. 销售发票的开具、购买
  • 3. 负责公司固定资产台账登记和管理
  • 4. 协助会计做好各种帐务的处理工作
  • 5. 完成上级交给的其它事务性工作
    • 岗位要求：
  • 1. 会计、财务等相关专业大专以上学历，有会计从业资格证书,有2年以上相关工作经验
  • 2. 了解国家财经政策和会计、税务法规，熟悉银行结算业务
  • 3. 熟练操作金蝶财务软件、OFFICE办公软件
  • 4. 善于处理流程性事务、良好的学习能力、独立工作能力和财务分析能力
  • 5. 工作细致，责任感强，良好的沟通能力、团队精神

会计主管

工作职责：
  • 1. 负责编制及组织实施财务预算报告，月、季度、年度财务报告
  • 2. 负责公司全面成本核算，会计核算、分析等基础工作
  • 3. 负责公司资金、资产的管理，固定资产的核算
  • 4. 负责公司产品仓库的监督和管理
  • 5. 组织实施公司于银行、工商、税务等工作的开展
  • 6. 完成公司与总公司之间财务工作相关联络的工作
  • 7. 负责公司商务相关工作的监督与管理工作
岗位要求：
  • 1. 28岁以上，会计、财务相关专业全日制正规大专以上院校毕业，5年以上企业财务管理工作经验
  • 2. 有会计上岗证，有中级职称
  • 3. 熟悉国家财经法律法规、会计准则和企业财务制度、税收政策及相关账务处理方法
  • 4. 熟悉金蝶软件操作、会计核算及税收申报管理
  • 5. 有较强的财务风险控制能力和财务分析能力
  • 6. 有软件行业和上市公司工作经验优先考虑
  • 7. 已婚已育者优先

客户经理

工作职责：
  • 1. 负责客户开拓：发掘目标客户资源，找到目标客户关键联系人，创造商务沟通的机会和条件
  • 2. 理解客户需求：通过各种方式深入理解客户各方面的需求，包括但不限于业务需 求、应用场景等
  • 3. 制定客户方案：根据客户的需求和公司的产品功能制定最能打动客户的方案计划
  • 4. 完成商务谈判，达成销售协议，实现货款回收
  • 5. 客情关系维护：维护与既有客户的良好客情关系，提升客户忠诚度，形成长期稳定合作关系
岗位要求：
  • 1. 28岁以上，会计、财务相关专业全日制正规大专以上院校毕业，5年以上企业财务管理工作经验
  • 2. 有会计上岗证，有中级职称
  • 3. 熟悉国家财经法律法规、会计准则和企业财务制度、税收政策及相关账务处理方法
  • 4. 熟悉金蝶软件操作、会计核算及税收申报管理
  • 5. 有较强的财务风险控制能力和财务分析能力
  • 6. 有软件行业和上市公司工作经验优先考虑
  • 7. 已婚已育者优先

销售总监

工作职责：
  • 1. 协助决策层制定公司发展战略，负责短期及长期的公司业务，对公司各阶段的销售目标的达成负责，高层管理职位
  • 2. 负责策划并带领执行公司年度市场工作目标和营销方案
  • 3. 负责开拓市场业务，进行相关的公司代理及产品宣传
  • 4. 分析和预测零售市场、把握大数据及零售行业互联网化市场的趋势，为决策提供准确的相关信息
  • 5. 统筹和建立公司各地办事处的销售管理事项，指导销售团队开拓市场
  • 6. 培养建立一支优秀凝聚力强的的营销团队
岗位要求：
  • 1. 5年以上的同岗位营销管理经验，擅长销售管理和营销策划
  • 2. 优秀的领导能力及影响力、判断与决策能力，人际沟通能力，谈判能力，计划和执行能力
  • 3. 市场营销管理类或软件相关专业本科以上学历，一定的英语读写和口语能力
  • 4. 拥有积极、健康的心态，抗压能力强
  • 5. 具备对市场以及行业行情敏锐的观察及判断能力，提前预备应对各种复杂情况的能力

售前工程师（数据及应用产品方向）

工作职责：
  • 1. 配合销售人员与客户沟通，了解客户的业务需求
  • 2. 协助销售，编写项目方案，并对客户进行产品及方案的讲解
  • 3. 负责公司产品案例、技术文档以及标书等对外材料的编制
  • 4. 与客户沟通，提供客户培训和产品交流工作
岗位要求：
  • 1. 大专及以上学历
  • 2. 思维逻辑清晰，方案撰写能力强
  • 3. 形象良好，沟通和演讲能力强
  • 4. 熟练使用PPT，word等办公软件
  • 5. 有零售、互联网、数据分析等相关领域经验者佳

产品经理

工作职责：
  • 1. 负责面向实体零售的数据应用解决方案、产品规划
  • 2. 挖掘零售各行业客户的数据应用需求，设计各行业数据解决方案和产品
  • 3. 组织和推动核心开发团队，快速完成解决方案产品化工作
  • 4. 面向销售及售前技术支持提供必要的培训及支持工作
岗位要求：
  • 1. 行业背景

    • * 具备O2O/零售/大数据/精准营销等任一相关方面的项目经验，理解行业业务，了解行业IT系统的功能及特点，对这些行业未来的业务及数据应用场景有理解、有想法，在相应行业从业1-2年以上

    * 具备上述行业数据统计、分析/应用产品设计、应用推广经验优先

  • 2. 专业背景

    • * 有数据/应用产品或互联网数据/应用产品设计、推广经验

    * 具备较强的数据分析能力、逻辑思维能力、沟通能力，能够有效与团队配合推动业务发展

    * 有数据分析挖掘或一定研发技术背景者为佳

    * 熟练使用Axure，Mindmanage等产品原型设计、思维导图工具，并能在每个研发周期出具相应的产品需求文档
